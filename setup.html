Langkah 1: Setup Google Apps Script

1 Buat Project Apps Script

  1. Buka script.google.com
  2. Klik "New Project"
  3. Ganti nama project menjadi "Sistem Tugas Voice"
  4. Hapus kode default dan paste kode dari file Google Apps Script - Updated Version

2 Jalankan Fungsi Setup

  1. Di Apps Script editor, pilih fungsi setup dari dropdown
  2. Klik tombol Run (‚ñ∂Ô∏è)
  3. Saat diminta permission, klik "Review Permissions"
  4. Pilih akun Google Anda ‚Üí "Advanced" ‚Üí "Go to [Project Name] (unsafe)"
  5. Klik "Allow"
  6. Tunggu sampai selesai (cek di Execution Log)
üìù Catat Info Berikut: Setelah setup berhasil, Apps Script akan menampilkan:
  • Spreadsheet ID - Copy dan simpan
  • Spreadsheet URL - Buka untuk melihat database
  • Folder URL - Folder untuk audio files

3 Update Spreadsheet ID

  1. Copy Spreadsheet ID dari execution log
  2. Di Apps Script editor, cari baris SPREADSHEET_ID: ''
  3. Paste ID di antara tanda petik: SPREADSHEET_ID: 'YOUR_ID_HERE'
  4. Klik Save (Ctrl+S)

4 Deploy sebagai Web App

  1. Klik tombol "Deploy" ‚Üí "New deployment"
  2. Klik icon ‚öôÔ∏è ‚Üí Pilih "Web app"
  3. Isi form:
    • Description: Production
    • Execute as: Me
    • Who has access: Anyone
  4. Klik "Deploy"
  5. Copy Web app URL yang muncul
‚ö†Ô∏è Penting: Web app URL berbentuk: https://script.google.com/macros/s/ABC.../exec
Simpan URL ini, akan digunakan di langkah selanjutnya.

Langkah 2: Konfigurasi HTML Files

5 Update Config di murid.html

  1. Buka file murid.html
  2. Cari baris: APPS_SCRIPT_URL: 'https://script.google.com/...'
  3. Ganti dengan Web app URL yang baru Anda copy
  4. Simpan file
const CONFIG = { APPS_SCRIPT_URL: 'PASTE_URL_ANDA_DI_SINI', CLASS_CODES: { '7A': 'KELAS7A2024', // ... dst } };

6 Update Config di guru.html

  1. Buka file guru.html
  2. Cari baris: APPS_SCRIPT_URL: 'https://script.google.com/...'
  3. Ganti dengan Web app URL yang sama
  4. Opsional: Ganti GURU_PASSWORD dengan password Anda sendiri
  5. Simpan file

7 Customize Kode Akses Kelas

  1. Di murid.html, edit object CLASS_CODES
  2. Ganti kode default dengan kode yang Anda inginkan
  3. Berikan kode ini kepada murid di setiap kelas
üí° Tip Keamanan: Gunakan kode yang unik dan kompleks, misalnya:
'7A': 'MTK7A_2024_UNIK123'

Langkah 3: Upload ke Web Hosting

8 Pilih Hosting Platform

Pilihan A: GitHub Pages (Gratis, Recommended)

  1. Buat repository baru di GitHub
  2. Upload 5 files: murid.html, guru.html, setup.html, manifest.json, service-worker.js
  3. Rename murid.html menjadi index.html
  4. Settings ‚Üí Pages ‚Üí Source: main branch
  5. URL akan jadi: https://username.github.io/repo-name/

Pilihan B: Netlify/Vercel (Gratis)

  1. Drag & drop folder ke Netlify/Vercel
  2. Otomatis deploy dengan custom domain

Pilihan C: Web Server Sendiri

  1. Upload files ke public_html atau www folder
  2. Pastikan HTTPS aktif

üîß Troubleshooting

‚ùå Error: "Sheet tidak ditemukan"

Solusi:

  • Pastikan fungsi setup() sudah dijalankan
  • Cek apakah Spreadsheet ID sudah benar di Apps Script
  • Jalankan fungsi testConfig() untuk validasi

‚ùå Submission tidak masuk ke Spreadsheet

Solusi:

  • Cek Console browser (F12) untuk error messages
  • Pastikan Web App URL benar di HTML files
  • Pastikan Apps Script di-deploy dengan "Who has access: Anyone"
  • Cek Execution log di Apps Script untuk error

‚ùå Audio tidak ter-upload

Solusi:

  • Cek ukuran file, max 10MB
  • Pastikan permission Google Drive sudah di-grant
  • Cek apakah folder "Rekaman Tugas Voice" ada di Drive

‚ùå Dashboard guru tidak load data

Solusi:

  • Buka Apps Script URL langsung: YOUR_URL?action=getSubmissions
  • Harus return JSON dengan data submissions
  • Jika error 403, cek deployment permissions
